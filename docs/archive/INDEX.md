## Archive Policy

### What Gets Archived

- **Progress tracking docs** — after feature completion
- **Superseded specs** — when replaced by newer versions
- **Legacy prompts** — from completed features
- **Old CLAUDE.md versions** — when significantly revised
- **Original source docs** — when consolidated into new docs

### What Stays Active

- **PROMPTS.md** — kept in feature folders as reusable templates
- **SPEC.md, DATA_MODEL.md, IMPLEMENTATION.md** — permanent reference
