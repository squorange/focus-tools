# Product Requirements Document: Focus Tools

## User Profile
- Neurotype: ADHD/AuDHD
- Primary Challenges: Task memory, prioritization, executive function, task initiation
- Current Pain Points: Fragmented task tracking across multiple tools

## Core Problem
Managing tasks across multiple tools (iOS Reminders, Notion, whiteboards, sticky notes) leads to:
- Lost tasks and forgotten commitments
- Difficulty reprioritizing when situations change
- Partner frustration acting as external executive function
- Overwhelming task pile-up

## Product Goals
1. Create clear mental model of all obligations
2. Reduce cognitive load of tracking/remembering
3. Support task completion, not just tracking
4. Remove partner reminder burden

## Key Features (MVP)
- Spatial task visualization with multiple "lenses"
- AI body double mode for task support
- Natural language task input
- Smart prioritization
- Daily focus sessions

## Success Metrics
- Daily active use without forcing
- Tasks completed vs deferred
- Partner satisfaction improvement
- Clear understanding of obligations
